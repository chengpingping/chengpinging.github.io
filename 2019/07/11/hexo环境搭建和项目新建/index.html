hexo环境搭建和项目新建

Posted on 2019-07-11

hexo是一个博客框架，使用Markdown语法编写内容，在官方网站中提供了很多的主题，可以美化你的博客界面。
官网：https://hexo.io/zh-cn/ （中文）

安装Git和Node.js

Git和nodejs为hexo框架提供环境
Git下载地址：https://git-scm.com/downloads
NodeJS下载地址：http://nodejs.cn/download/
根据自己电脑情况，下载对应的版本。安装步骤简单，没有什么特别的注意事项，如果不想安装到C盘可以换到其他盘！
检查是否安装成功：
打开dos界面：win+R/直接搜索‘DOS’或者‘命令’


Git安装成功后，鼠标右键会有Git GUI Here和Git Bash Here两个选项

安装hexo

右键–>点击Git Bash Here，会弹出一个窗口，然后输入:npm install -g hexo-cli，等待一会。。。

开始新建自己的博客

1.hexo安装成功后，选择/新建一个自己放博客项目的文件夹；
2.右键右键–>点击Git Bash Here，输入：hexo init(初始化自己的环境)，可能会等待一会儿，也可能会失败（外网），成功后会在当前目录下会出现一些文件和文件夹；
3.输入：hexo generate/hexo g(生成静态界面)
4.输入：hexo server/hexo s(启动本地服务器)，此时你就可去访问你的第一个博客网站了：浏览器输入 http://localhost:4000 ；
5.输入：hexo new （这里输本篇博客的名字），新建自己的博客内容，会在：当前目录\source_posts 目录下生成一个后缀为.md的文件（Markdown文件）
6.编辑文件，可以用任意的文件编辑器编辑文档内容，比如：VSCode。Markdown是一种轻量级语言， 编写的文档可以导出 HTML 、Word、图像、PDF、Epub 等多种格式的文档。Markdown教程：https://www.runoob.com/markdown/md-tutorial.html
7.保存文档刷新页面，你就可以看见自己的博客内容。

将自己的博客发布到云服务器上

github是一个基于git的代码托管平台，付费用户可以建私人仓库，我们一般的免费用户只能使用公共仓库，也就是代码要公开，对于学习者这就够了。放在托管平台上的好处是任何人都可以通过网络访问到你的网站。

在GitHub上申请一个账号

官网：https://github.com/
注册成功登陆以后，你就可以新建一个仓库：Create a New Repository。（填上所有信息点击creat）
注意命名：GitHub账户名.github.io（严格按照此格式）

将自己的博客发布到GitHub上

在项目法人更目录下有一个_config.yml文件,修改Deployment下面的repo:https://GitHub账户名.github.io (这里填仓库地址)

上传代码到GitHub

1.安装：在Git窗口中输入 npm install hexo-deployer-git –save
2.执行hexo deploy/hexo d(输入自己的GitHub账号和密码)，你可以看见你的文件正在上传。
3.访问自己的仓库，浏览器中输入： https://GitHub账户名.github.io，就可成功访问自己的博客了。
